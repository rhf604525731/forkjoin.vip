Hexo简介

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

环境准备

Hexo是基于Node.js开发的，首先需要在个人电脑上安装Node.js程序,下载自己电脑系统对应位数(64位),(32位)的最新版本。因为Hexo生成的静态HTML文件我们要发布到github上,所以Git 程序也是要安装的。

git安装

64-bit Git for Windows Setup

双击exe安装程序，一步一步执行到最后安装完成，推荐使用默认配置项。
安装成功后会出现在“开始”–>”所有程序”–>”Git”菜单中。如图：

打开 Git CMD 或者 Git Bash 输入: git --version

如果能正常执行并返回git的版本号，说明git的安装没有问题

nodejs安装

Node.js for Windows (x64)

安装Hexo

如果您的电脑中已经安装上述必备程序，那么恭喜您！接下来只需要使用 npm 即可完成 Hexo 的安装

打开 Git Bash 输入：install -g hexo-cli

我个人喜欢这种 linux shell 风格的命令行窗口。
